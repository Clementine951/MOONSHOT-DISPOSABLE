The Features

App Clip Event – Capture or view photos in a simplified interface, perfect for guests.

Home – Create a new event or join an existing one with a simple tap.

Create Event – Set up your event details, choose when it starts, how long it lasts, and how photos are revealed.

Event Overview – See key info: event name, participants, and a countdown until the event ends.

Capture Photos – Snap ephemeral pictures. Retake or save them, and watch your remaining photo count.

Gallery – Browse personal or shared photos. Tap to enlarge and enjoy the event’s images.

Join Event – Enter the event ID to hop into an existing party or gathering.

Settings – Manage your profile, read policies, or adjust preferences for a tailored experience.

App Clip Card –What are the information and possibilities

App Clip Join – Quickly see the event’s name and details at a glance.
